### **用户端使用手册**
##### 以下将运行的微信账号简称为微信

#### 服务指南
向微信发送各种关键词即可获得反馈，发送菜单可呼出提示菜单

#### 服务菜单一览
* 获取最新微博消息请发送【最新微博】
* 回复【关闭聊天机器人】可关闭自动聊天功能（回复【开启聊天】可开启）
* 回复【最新微博+条数（数字）  eg：最新微博+3】可获得符合数量的最新消息
* 回复【搜索推特用户+关键词】搜索用户锁定id~
* 想知道谁刚刚说了什么可以回复【最新推特+用户ID】
* 获取指定的最新推特可回复【最新推特+用户ID+数量（数字）不大于5】
